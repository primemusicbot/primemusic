# 🎬 My Video Music Bot

Telegram Voice Chat के लिए Full HD Video Music Bot

## ✨ Features
- 🎥 YouTube Video Playback in VC
- 🎵 Audio Only Mode
- ⏸ Pause/Resume/Stop Controls
- 📋 Queue System

## 📱 Commands
| Command | Description |
|---------|-------------|
| `/video <song>` | Play video |
| `/audio <song>` | Play audio only |
| `/pause` | Pause playback |
| `/resume` | Resume playback |
| `/stop` | Stop playback |
| `/skip` | Skip to next |
